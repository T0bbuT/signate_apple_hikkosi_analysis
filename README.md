# 引っ越し需要予測のための機械学習モデル
## 概要
[SIGNATEのコンペ「アップル 引越し需要予測」][1]を題材としたデータ分析。

引越し業者における時期ごとの最適な価格設定のため、機械学習を利用した受注件数の予測モデルを開発  
最終的なモデルは
- 長期的な件数変動のトレンドを捉えるモデル
- 日々の細かい件数の変動を捉えるモデル

2つの基本モデルを組み合わせて構築
